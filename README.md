# victorhahn-skills

Personal Claude Code skills and plugins.

## Installation

```bash
# npm
npx skills add victorhahn/victorhahn-skills

# pnpm
pnpm dlx skills add victorhahn/victorhahn-skills
```

## Skills

### `org-context`

Builds and queries a structured knowledge base for a GitHub org or repo bundle — tech stack, domains, runbooks, cross-repo connections, team ownership, and release process. The "onboarded engineer" context layer: build it once, use it everywhere.
